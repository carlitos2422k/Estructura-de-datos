Manual de Ayuda - Sistema de Gestión de Biblioteca

1. Introducción

Bienvenido al Sistema de Gestión de Biblioteca. Esta aplicación permite administrar libros, usuarios y préstamos de una biblioteca de manera eficiente. Utiliza estructuras de datos avanzadas como árboles AVL para libros, tablas hash para usuarios y listas doblemente enlazadas circulares para el historial de préstamos.

El sistema ofrece dos tipos de usuarios:

  • Usuarios regulares: Pueden solicitar préstamos, devolver libros y ver recomendaciones personalizadas.
  • Administradores: Tienen acceso completo a la gestión de libros, usuarios y sistema de préstamos.

2. Menú Principal

Al iniciar la aplicación, se muestra el menú principal con las siguientes opciones:

Opción Descripción
Iniciar sesión Permite a usuarios existentes acceder al sistema con su ID y contraseña.
Registrar nuevo usuario Permite crear una cuenta de usuario regular nueva.
Registrar nuevo administrador Permite crear una cuenta de administrador nueva (solo disponible en modo demo).
Mostrar todos los usuarios registrados Muestra la lista de todos los usuarios registrados en el sistema.
Ayuda Muestra este manual de ayuda.
Salir Cierra la aplicación.

Nota: Puedes navegar por los menús utilizando las teclas de flecha arriba/abajo y presionar Enter para seleccionar una opción. La tecla Esc te permite volver al menú anterior.

3. Registro e Inicio de Sesión

3.1 Registro de Usuario

Al registrar un nuevo usuario, deberás proporcionar la siguiente información:

  • ID de Usuario: Un identificador único para tu cuenta.
  • Nombre: Tu nombre completo.
  • Contacto: Una dirección de correo electrónico válida.
  • Contraseña: Una contraseña de al menos 6 caracteres.

3.2 Inicio de Sesión

Para iniciar sesión, necesitarás:

  • Tu ID de usuario
  • Tu contraseña

Usuarios de prueba por defecto:

  • Usuario regular: ID: U001, Contraseña: user123
  • Administrador: ID: A001, Contraseña: admin123

Una vez iniciada la sesión como usuario regular, tendrás acceso a las siguientes opciones:

Opción Descripción
Ver mis préstamos activos Muestra los libros que tienes prestados actualmente.
Ver mi historial de préstamos Muestra todos los préstamos que has realizado, tanto activos como devueltos.

6. Gestión de Libros

6.1 Añadir un Libro

Para añadir un nuevo libro al sistema, deberás proporcionar:

  • Código: Un identificador único en formato LXXX (donde X son números), por ejemplo: L001.
  • Título: El título del libro.
  • Autor: El nombre del autor.
  • Ubicación: La ubicación física del libro en la biblioteca.
  • Género: El género literario al que pertenece.
  • Copias disponibles: El número de ejemplares disponibles.

6.2 Modificar un Libro

Permite actualizar cualquier información de un libro existente. El código no puede ser modificado, ya que es el identificador único.

6.3 Eliminar un Libro

Permite eliminar un libro del catálogo. No se pueden eliminar libros que tengan préstamos activos.

7. Sistema de Préstamos

7.1 Solicitar un Préstamo

Para solicitar un libro en préstamo:

  1. Selecciona "Solicitar préstamo" en el menú de usuario.
  2. Verás una lista de libros disponibles.
  3. Ingresa el código del libro que deseas.
  4. Confirma la solicitud.

El sistema registrará tu préstamo y actualizará las copias disponibles del libro.

7.2 Devolver un Libro

Para devolver un libro:

  1. Selecciona "Devolver libro" en el menú de usuario.
  2. Verás una lista de tus préstamos activos.
  3. Ingresa el ID del préstamo que deseas devolver.
  4. Confirma la devolución.

7.3 Multas por Retraso

Si devuelves un libro después de la fecha límite, se generará una multa. El sistema calcula las multas a razón de $5 por cada día de retraso.

8. Sistema de Recomendaciones

El sistema cuenta con un algoritmo de recomendación que sugiere libros basándose en:

  • Historial personal: Los géneros que has leído anteriormente.
  • Popularidad: Libros frecuentemente solicitados por otros usuarios.
  • Relaciones entre géneros: Géneros relacionados con tus preferencias.
  • Novedades: Libros que no has leído antes.

8.1 Configuración del Sistema de Recomendaciones

Los administradores pueden ajustar los parámetros de recomendación:

  • Configurar pesos manualmente: Ajustar la importancia de cada factor.
  • Establecer configuración predeterminada: Restaurar los valores por defecto.
  • Ver configuración actual: Mostrar los valores actuales.
  • Regenerar índice de recomendaciones: Actualizar los datos utilizados para las recomendaciones.

9. Respaldo y Recuperación

El sistema realiza backups diarios automáticos de la información de préstamos. Al iniciar, puedes elegir cargar un backup específico si es necesario.

9.1 Cargar un Backup

Al iniciar el sistema, tienes la opción de cargar un backup anterior. Esto puede ser útil si necesitas recuperar información de días anteriores.

10. Estructuras de Datos Utilizadas

El sistema utiliza estructuras de datos eficientes para garantizar un buen rendimiento:

  • Árbol AVL: Para almacenar y buscar libros de manera eficiente.
  • Tabla Hash: Para gestionar usuarios con acceso rápido.
  • Lista Doblemente Enlazada Circular: Para manejar el historial de préstamos.

11. Solución de Problemas

11.1 No puedo iniciar sesión

Verifica que estés usando el ID y contraseña correctos. Si olvidaste tu contraseña, contacta con un administrador.

11.2 No puedo solicitar un préstamo

Asegúrate de que:

  • El libro que deseas tenga copias disponibles.
  • El código del libro sea correcto.
  • No tengas multas pendientes que bloqueen nuevos préstamos.

11.3 No veo mis recomendaciones

Para recibir recomendaciones personalizadas, debes tener al menos un préstamo registrado en tu historial. Si eres un usuario nuevo, primero solicita algunos libros.
